03. Gestión Post-Entrega

Una vez entregado el producto, comienza la fase de mantenimiento:

  • Incidencias (Bugs): Errores que el sistema presenta frente a lo especificado originalmente.
  • Nuevos Desarrollos (Features): Evolución del sistema ante cambios en el entorno o necesidades del usuario.

"El software nunca se termina, solo se abandona."
