02. El Servidor HTTP desde la Base (C++)

Un servidor HTTP en C++ (usando `cpp-httplib`) opera directamente sobre POSIX Sockets. La lógica de fondo implica:

  • TCP Three-way Handshake: El servidor debe gestionar el establecimiento de la conexión antes de recibir un solo byte de HTTP.
  • Parsing Serial: Se recibe un flujo de bytes que debe ser parseado según la gramática de la RFC 7230.
Referencia Académica:

Stevens, W. R. (1998). UNIX Network Programming. Prentice Hall. (Buscar en Google Académico para entender la gestión de descriptores de archivos y sockets).
